# v0.1.1 Release Notes

Token Cost App - OC Codex Ver 的第一个 patch 版本。核心变更：**计费方案管理**、**中英双语本地化**、**token 计算修正**。

---

## 计费方案管理

设置页新增「计费方案」区域，为每个数据源提供官方订阅 / Token Plan 预设档位 + 自定义 USD 月费：

| Provider | 档位 |
|----------|------|
| **OpenCode** | Go ($10/月)、Zen (按量计费) |
| **ChatGPT / Codex** | Plus ($20/月)、Pro ($200/月)、Business Codex (按量计费) |
| **MiniMax** | Starter ¥29、Plus ¥49、Max ¥119、Plus 极速 ¥98、Max 极速 ¥199、Ultra 极速 ¥899 |
| **Xiaomi MiMo** | Lite ¥39/$6、Standard ¥99/$16、Pro ¥329/$50、Max ¥659/$100，含年付方案 |

- 自定义 USD 月费：应对官方价格变更
- 按量计费 / Contact-Sales 档位作为说明项，可通过自定义月费纳入总成本
- 内置只读计费参考文档（设置页 📄 按钮）
- 旧版偏好自动兼容迁移

## 中英双语本地化

所有 UI 文案接入 `AppLocalization` 层，支持中文/英语实时切换。菜单、设置、状态提示、Codex 会话明细、排序表头等 350+ 处文案均可切换。

## Token 计算修正

- **修复 actual token 重复扣减缓存 bug**：DashboardAnalytics、TokenDatabaseClient、Models 中 `actualTokens` 公式从错误的 `max(input - cacheRead - cacheWrite, 0)` 修正为 `input + output + reasoning`
- **修复 totalActualInputTokens 遗漏**：Models.swift 中的 totalActualInputTokens 计算属性此前仍执行旧公式，导致 TotalView 实际输入 token 偏低

## 文档

新增 `docs/功能模块关联清单.md`，完整记录所有功能、模块文件、数据管线与交叉依赖关系。修改任何功能前应先阅读此文档。

## 其他

- 项目目录标准化为 SPM 根路径结构
- CI/CD workflow（ci.yml、release.yml）
- 本地化资源打包（Package.swift `.process("Resources")`）
- 单元测试覆盖：旧配置解码、计费覆盖、分析引擎注入、actualInputToken 回归
